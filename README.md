# 💰 Finance Tracker App

## 📌 Overview

**Finance Tracker** is a user-friendly Android application developed using **Kotlin** and **Android Studio**.  

The app helps users efficiently manage their daily income and expenses by allowing them to record, update, delete, and track financial transactions in an organized manner.

Designed with simplicity and usability in mind, the application provides a clean interface and smooth user experience to support better personal financial management.

---

## ✨ Core Features

### 📝 Transaction Management (Full CRUD)
- Add new income and expense records  
- View transaction history  
- Update existing transactions  
- Delete unwanted records  

---

### 📊 Expense Tracking
- Categorize transactions  
- Monitor total income and total expenses  
- View real-time balance updates  

---

### 🔔 Smart Organization
- Simple and intuitive UI  
- Efficient data persistence  
- Structured financial record management
